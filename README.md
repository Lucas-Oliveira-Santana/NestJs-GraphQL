## Description

A CRUD ant authentication users with NestJs, GraphQL

# For Test

## Clone Repository

```bash
$ git clone https://github.com/Lucas-Oliveira-Santana/NestJs-GraphQL.git
```

## Running the app

```bash
# up Database(Postgres) and API
$ docker-compose up

```

## Test

```bash
# unit tests
$ npm run test

```
